---
title: "Dashboards"
---

The **Dashboards** section allows you to create custom dashboards tailored to specific use cases such as product monitoring, experimentation, customer support, or executive reporting.

### Creating dashboards

- Navigate to **Dashboards** and click **Create Dashboard**
- Provide a name and optional description
- Add one or more **saved charts** to the dashboard

### Using dashboards

- All configured dashboards are listed under the Dashboards section
- Clicking a dashboard opens a consolidated view of all associated charts
- A chart can be added to multiple dashboards

Dashboards enable teams to group related insights and reuse charts across different analytical views.

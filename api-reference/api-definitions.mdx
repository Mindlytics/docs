---
title: "Available APIs"
---

Mindlytics APIs are designed for the applications developers to meet the following objectives,

1. **Manage user identities:** Identify users, enrich user identity with user traits, manage user's unidentified and identified states using aliasing.
2. **User Session Tracking:**  Track user sessions.
3. **Conversation Tracking:** Track beginning and end of a conversation with granular turn by turn tracking.
4. **Custom Tracking:** Track user's explicit in-app actions or AI's responses through custom, developer defined event names.
